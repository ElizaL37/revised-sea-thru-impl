# revised-sea-thru-impl
An Implementation of Sea-Thru a physics-based underwater image enhancement algorithm, this implementation has been designed with, the best of my ability, to replicate the method described by the original researchers, with exceptions for increased efficiency and robustness across varying scenes. The provided solution is an initial step for a 3D modelling pipeline for SHM.
